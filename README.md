# kivicaptial-assignment
# Overview
Its a simple Rock Paper Scissor game.Contains two files one in c++ and another one in python both do the same job.

# Working
Its you vs computer. Computer randomly picks one(Rock,Paper,Scissor) and for Users choice it takes input from the user infinite times and prints the result.

# Usage 
Run any one of two files. Type your input(Rock,Paper,Scissor) and it displays the result.
